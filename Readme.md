# Easy MQTT

**Easy MQTT** enables developers to quickly and easily establish MQTT connections and initiate communication. At its core, the asset relies on a DLL that wraps the widely used and well-known .NET library [MQTTnet](https://github.com/chkr1011/MQTTnet), while providing additional features to simplify development for both beginners and advanced users.

---

## Key Features

- **Quick MQTT Client Setup:** Easily create and configure MQTT clients.  
- **Fluent Subscription Builder:** Register handlers with an intuitive builder pattern.  
- **Route-Based Dispatching:** Support multiple handlers per topic with router-managed lifecycle.  
- **Simple Publish/Subscribe:** Focused surface for JSON payloads or plain strings.  
- **Automatic Reconnection:** Configurable timeout for automatic reconnects.  
- **Flexible Connections:** Support for secure or unsecure TCP/WebSocket connections with optional credentials (username/password).
